# Città Semplice - NextJS - TypeScript

App per gestione dei servizi online a istanza di parte del Comune di Catania - Città Semplice.

## Regole Critiche
- Analizza la cartella @E:\workspaces\citta-semplice-next-gen\frontend\citta-semplice-3, contiene una soluzione Angular di cui voglio utilizzare solo i layout delle pagine
- analizza la parte di backoffice di questa applicazione nella cartella @E:\workspaces\citta-semplice-2026\citta-semplice-office, usa le stesse entità i database, 
- analizza la versione precedente del frontend di questa applicazione in Angular 8 nella cartella @E:\workspaces\catania-semplice\citta-semplice\frontend\citta-semplice
- analizza la versione precedente del backend di questa applicazione in Java 8/Spring Boot nella cartella @E:\workspaces\catania-semplice\citta-semplice\backend\citta-semplice
- utilizza la versione precedente come schema per capire qual'è l'obbiettivo da raggiungere

## Stack Tecnologico
- NextJS 16.1+
- TypeScript
- zod per la validazione dei form
- bootstrap italia con il tema dei comuni
- componenti server con actions 
- componenti client per i form e le tabelle da popolare con fetch paginate
